# big_screen
数据大屏可视化

# 功能

便利性工具, 结构简单, 直接传数据就可以实现数据大屏

# 安装

```
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple flask
```

# 运行

```
cd big_screen;
python app.py;
```
